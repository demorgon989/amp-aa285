# AMP Template - America's Army Operations 2.8.5 Server

Custom CubeCoders AMP template for running an America's Army Operations 2.8.5 dedicated server through Wine.

## Release files

- `aa285-server.zip` - Core server install used by AMP.
- `aa285-maps.zip` - Optional extra map pack. Extract this over the server root if you want the remaining maps.

The core server archive currently includes:

- `Pipeline.aao`
- `SFhospital.aao`

## Notes

This package is intended for server use only, not as a full client install.

Default game port:

- UDP `1716`
